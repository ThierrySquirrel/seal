# 内存清理:

自动清理Windows系统内存.  
当你需要专注于使用Windows在多个地区工作时,Seal可以成为你的朋友.  
当你在全球旅行并需要在Windows上工作时,海豹可以成为你的朋友.  
当你在高压环境中工作,需要专注于使用Windows系统时,Seal可以成为你的朋友.  

Seals可以在Microsoft Windows系统架构的复杂内存模型中执行内存回收.  

海豹:擅长反步兵作战和爆破建筑物,但对装甲部队的影响较差.  
海豹:具有水下潜水,摧毁船只和直接摧毁建筑物的能力.
